Personal Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and professional experience. Built with clean design and smooth interactions to create an engaging online presence.

Features

Responsive Design - Looks great on all devices
Modern UI/UX - Clean and professional interface
Project Showcase - Display your work with descriptions and links
Contact Form - Easy way for visitors to get in touch
Fast Performance - Optimized for quick loading times
SEO Friendly - Built with best practices for search engines

Tech Stack

Frontend: HTML5, CSS3, JavaScript
Styling: Custom CSS (consider adding Tailwind CSS/Bootstrap if used)
Deployment: Netlify
Version Control: Git & GitHub
